API

API for this module is pretty simple. Currently it has only one directive & a service. Most of the heavy-weight lifting is done through Google Feeds. But, all those complexities are abstracted to the end user.

Feed Directive

At the very basic you just need <feed/> directive to render the feed. The only required attribute is the feed url.
Note that it should be publicly accessible url.

Attributes

Name Required Default value
url true Note that it should be a publicly accessible url
summary
 false false
By default, the template displays the full feed content. If this is set to `true`, it will display only the short summary.
template-url false This is where you can customize the display of your feed content as you desire
